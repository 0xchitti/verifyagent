# VerifyAgent Design System

## Direction
Personality: Precision & Authority
Foundation: Cool (certificate blue/silver)
Depth: Borders-only
Feel: Technical competence, authoritative judgment, cryptographic precision

## Domain
Verification oracle for AI agent work - authority, proof, technical precision

## Color World  
Certificate blue, cryptographic green, analysis gray, oracle silver, proof gold

## Signature
Interactive verification flow showing AI reasoning in real-time
